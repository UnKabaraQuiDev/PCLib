# PCLib Common

Shared helpers and core utilities used by other PCLib modules.

**Java version:** Java 8

## Maven

```xml
<dependency>
  <groupId>lu.kbra</groupId>
  <artifactId>pclib-common</artifactId>
</dependency>
````

## What it contains

This module contains general-purpose helpers that do not fit into a more specific module.

Main class:

* `lu.kbra.pclib.PCUtils`
