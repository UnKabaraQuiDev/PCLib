# PCLib AWT

Helpers for Java AWT drawing and image work.

**Java version:** Java 8

## Maven

```xml
<dependency>
  <groupId>lu.kbra</groupId>
  <artifactId>pclib-awt</artifactId>
</dependency>
````

## What it contains

This module includes:

* image drawing helpers
* polyline support
* brightness composite utilities

Main classes:

* `ImageDrawer`
* `Polyline`
* `BrightnessComposite`
