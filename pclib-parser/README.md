# PCLib Cache

Simple cache utilities with optional timed expiration. _Not actively maintained_

**Java version:** Java 8

## Maven

```xml
<dependency>
  <groupId>lu.kbra</groupId>
  <artifactId>pclib-cache</artifactId>
</dependency>
````

## What it contains

This module provides:

* cache entries
* cache lists
* timed cache entries
* timed cache lists
* scheduler-based timed cache lists

Main classes:

* `CacheEntry`
* `CacheList`
* `TimedCacheEntry`
* `TimedCacheList`
* `SchedulerTimedCacheList`
